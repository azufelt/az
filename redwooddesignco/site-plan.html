Typography

Heading Font: Playfair Display

I chose this for the Heading font because it has a sophisticated element. Being a serif font it lends toward a little bit more formal or high-end product. It subtly tells the user that this is a site and brand that is trustworthy.

Paragraph Font: Montserrat

Montserrat is a great pairing with Playfair Display. Because it is a sans-serif font it is easier to read for paragraphs and descriptions. Also the glyph or character size matches similarly to the heading font, reflecting a similar regalness. The juxtiposition of having a serif font for the heading and sans-serif for the paragraphs allows the user to easily distinguish between heading/description.
