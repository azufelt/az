Typography Design

Heading Font: Work Sans

I chose this because it has a more conservative feel for a sans serif typeface, which is inline with the target audience. It allows the emphasis to remain casual and friendly, while upholding a conservative feel which be appealling to the age demographics of the typical site visitor.

Paragraph Font: Heebo

I chose this because it has very similiar elements to the heading font but has simplified descenders, which will allow the body copy to be more easily viewed and consumed.
